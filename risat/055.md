# Artículo 55.

Compete a las administraciones desconcentradas jurídicas ejercer las atribuciones señaladas en las fracciones VII, VIII, IX, XI, XII y XIV del artículo 54 de este Reglamento.

Para el ejercicio de las atribuciones señaladas en este artículo, los titulares de las unidades administrativas serán auxiliados por los subadministradores desconcentrados jurídicos, jefes de departamento y demás personal que se requiera para satisfacer las necesidades del servicio.
